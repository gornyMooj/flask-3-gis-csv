Provide the csv to get back spatial data

💡 I don't save files on the server

How it works:

  • upload a csv file with coordinates broken down into 'X' and 'Y' columns.
  • it is assumed that the points that you are uploading are in the EPSG:4326 coordinate system
  • select the desired data format
  • click "Download" to download the data in the selected format

The app utilises the following tools:

  • Flask
  • Pandas
  • Geopandas
  • Liflet

🌍Here is a map viewer you can use to quickly generate a CSV file of points.
